# React の始め方（Vite 編）
本プロジェクト `with-vite` は、React の基本的な概要と動作を理解するためのサブプロジェクト。
このファイルがある階層で、以下の手順を実施していくことで、サンプルコードと同様の成果物が作れる。

`with-vite` は、主に **ツール利用による環境構築の容易さ** と **React の基本的な書き方** を説明する。
